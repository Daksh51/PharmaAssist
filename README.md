# PharmaAssist
A modern desktop medicine information application built with **Python** and  **Tkinter**
---

## Features
- Search medicine instantly
- Search using brands
- Typo correction with smart suggestions
- Add, view and remove favourites
- Persistent favourites using JSON
- Copy medicine details to clipboard
- Modern desktop GUI
- Fast and lightweight
---

## Build With
- Python 3.13
- Tkinter
- JSON
